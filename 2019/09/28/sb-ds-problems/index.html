傻逼数据结构题口胡记

持续更新 | 发表于 2019-09-28 更新于 2019-10-22 分类于 Solutions 阅读次数： Valine：

这里是一些数据结构傻逼题的口胡。主要是为了锻炼代码能力。

SPOJ GSS7

题意

给一棵树，支持链上覆盖，链上 GSS。

\(n,m\le10^5\)。

解法

为了防止忘记怎么打树剖和线段树就没有写 LCT 版本 23333

按照套路，记录区间和、最大前缀和、最大后缀和和 GSS。

注意这题貌似可以不选数字，pushdown 的时候不要想太多……

「SCOI 2010」序列操作

题意

给一个 01 序列，支持区间覆盖、区间取反、区间和、查询区间极长全 1 段。

\(1\le n,m\le10^5\)。

解法

维护 0 和 1 的数量、最长前缀、最长后缀、极长全 0/1 段。注意标记的处理方式：

  • 区间覆盖的时候，清空区间取反的标记。
  • 区间取反的时候，如果有取反标记，那么清空取反标记；否则如果有区间覆盖标记，那么翻转区间覆盖标记。

P2787 语文1

题意

给一个字母串，支持查询一种字符的出现次数、区间覆盖、区间排序。

\(1\le n,m\le5\times10^4\)。

解法

线段树每个点记录区间中每种字母的出现次数，那么 1、2 操作就很显然了。

然后根据套路，可以将 3 操作转换成 \(O(|\Sigma|)\) 次 1、2 操作，然后就做完了。

建议只开一棵线段树，每个节点存 26 个数，这样可以显著地减小常数。虽然常数还是被爆踩了

SPOJ GSS6

题意

给一个序列，支持插入删除修改区间 GSS。

\(1\le n,m\le10^5\)。

解法

直接用 Splay 维护就完事了。此题严格弱于「NOI 2005」维护数列。注意细节。

CF438D

题意

给一个序列，支持区间加、区间取模、单点修改。

\(1\le n,m\le10^5\)。

解法

注意到对于 \(x<y\)，一定有 \(y\bmod x<\frac y2\)。所以一个数一直取模的话，最多变 \(\log\) 次。

考虑用类似区间开方、区间求和的做法。记录区间最大值。如果最大值小于模数，那么不递归，否则直接暴力递归到叶子。

复杂度盲目分析，感性理解即可

GSS5

题意

给一个序列，支持查询左端点在 \([x_1,y_1]\)，右端点在 \([x_2,y_2]\) 下的 GSS。满足 \(x_1\le x_2,y_1\le y_2\)，但不保证区间不相交。

解法

不相交的话就是傻逼题。相交可以分成三种情况：

  • \(x\in[x_1,x_2),y\in[y_1,x_2]\)
  • \(x\in[x_1,y_1],y\in(y_1,x_2]\)
  • \(x,y\in[x_2,y_1]\)

然后分别查询就没了。

「SHOI 2015」脑洞治疗仪

题意

一个序列，支持区间置 0；把一个区间里的 1 依次填到另一个区间里的 0 上，然后清空这个区间；查询区间最长连续 0 段。

解法

操作 1 操作 3 直接做。

操作 2 在线段树上二分即可。
