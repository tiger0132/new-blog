一些常见套路

持续更新 | 发表于 2019-08-22 更新于 2019-08-29 分类于 Notes 阅读次数： Valine：

里面的套路大概都比较基础。持续更新……

位运算

按位运算的每一位都是独立的。有时候可以分开考虑。（经典套路）

一串连续的位运算可以简化成 一次或和一次异或 或者 一次与和一次异或。（CF878A P5354）

注：P5354 的常见做法不需要这个性质，但是本蒟蒻的一种做法使用了这个性质。

一个 \([0,2^n)\) 范围内的二进制数组的前缀或和前缀与最多有 \(n+1\) 种取值。（N933B）

二元的操作序列转换成二进制串可能有意想不到的效果。（P4424）

多个元素的 奇偶性 和 存在性 可以压缩成单个二进制数。（经典套路）

图论

在连通性问题中，一个点集中的所有点互相连边，可以转化为，一个点集中的点连向一个虚点。使用线段树优化建图可以进一步优化边数。

形如 \(f_x=\min_{y\in S_x}\{f_y+w(x,y)\}\) 且有后效性的 DP 可以使用最短路转移。（经典套路）

同余可达性问题（其实不只是可达性）可以转化为最短路。

DP

如果有多个决策点对答案的贡献是相同的，或者只有少部分决策点对答案有贡献，且这些决策点有性质，可以考虑只转移它们，以优化复杂度。（N933B）

树

一棵树的子树在 DFS 序上体现为一个区间。（经典套路）

如果 \(m-n\) 的值非常小，那么可以考虑维护它的一棵生成树

数据结构

维护一种信息的方式只有两种：拆成多个信息 或 考虑修改操作对信息的修改。（经典套路）

在与顺序无关的题中，修改可以转化成 删除后插入。（经典套路）

可以维护全局修改、全局查询、单点修改，但是不能维护区间的时候，可以考虑分块。（经典套路）

其它

最小化问题可以转化为枚举，如果有单调性可以转化成二分。（经典套路）
