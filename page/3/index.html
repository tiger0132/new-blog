「Internal」SPQR Tree 口胡记

沉底 | 发表于 2019-10-06 更新于 2019-11-17 分类于 Notes Valine：

SPQR Tree 是一种用来描述三连通结构的数据结构。

注意：本文中包含很多自行翻译的名词，切勿当真，感谢合作。

三连通分量

众所周知，点 / 边 双连通分量是指任意删除 一个点 / 一条边 后仍然连通的图。

定义 \(k\) 连通指，任意删除 \(k-1\) 个点 / 条边 后仍然连通的图。

也就是说，点三连通是指任意删除两点后仍然连通的图。

SPQR Tree 的结构

众所周知，圆方树（虽然只是一个误传的译名）有圆点和方点。

那么显然，SPQR Tree 有 S、P、Q、R 点。

  • S 点代表一个 环。这种情况和 串并联图 中的 串联组合 类似，即 S 代表 series。
  • P 点代表一个 偶极图，即两点之间连有多条重边的图。这种情况和 串并联图 中的 并联组合 类似，即 P 代表 parallel。
